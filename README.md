# ishigami_covparam_optim
Supplementary experiments for the paper “Goal-Oriented Lower-Tail Calibration of Gaussian Processes for Bayesian Optimization”.

This repository contains additional results on covariance hyperparameter optimization for the Ishigami function with two added input dimensions, for a total input dimension of d=6. We optimize the covariance hyperparameters, including the output scale and lengthscales, and compare the performance of tcGP and GP.
